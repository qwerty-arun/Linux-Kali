# Adding and Removing Software

## Search for a Package
- `apt-cache search snort`
- `apt-get install snort`
- `apt-get remove snort`
- `apt-get purge snort` to remove the config files at the same time as the package.
- `apt-get upgrade`

## List Kali repos
- `vim /etc/apt/sources.list` to list all the Kali repositories in the sources.list file.
